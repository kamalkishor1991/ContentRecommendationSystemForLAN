The goal of this system is:
To build a system which will overcome all limitations in LAN regarding the recommendation of distributed content.To minimize users efforts to provide feedback To maximize the users satisfaction by displaying appropriate content at the right time.System involves collection of user preferences for the available content and to learn user behaviour from that Provides recommendations to users on the basis of their behaviour & content popularity The system provides searching and sorting.To Implement this system is divided into 6 packages 1-distSystem 2-fileInfo 3-Log 4-search 5-utility 6-webServer
